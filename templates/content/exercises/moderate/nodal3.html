Question 3

Write the nodal equations for the network in Figure 18.27, which features a current-controlled current source (CCCS). Determine the final, simplified KCL expressions for both Node 1 and Node 2 in terms of the node voltages ($\mathbf{V}_1$, $\mathbf{V}_2$).

Figure 18.27: Nodal network containing a dependent current source controlled by branch current I'

Your Solution:

Enter the simplified KCL equation for Node 2.

💡 Hint 1: Expressing the Controlling Variable

The dependent source relies on $\mathbf{I}'$. Look at the diagram: $\mathbf{I}'$ is the current flowing from Node $\mathbf{V}_1$ to Node $\mathbf{V}_2$ through impedance $\mathbf{Z}_2$. Using Ohm's Law, this can be written as: $\mathbf{I}' = \frac{\mathbf{V}_1 - \mathbf{V}_2}{\mathbf{Z}_2}$.

💡 Hint 2: KCL at Node 2

When summing the currents leaving Node 2, remember to include the dependent source. The currents leaving are: $\frac{\mathbf{V}_2 - \mathbf{V}_1}{\mathbf{Z}_2}$ (going left), $\frac{\mathbf{V}_2}{\mathbf{Z}_3}$ (going down), and $k\mathbf{I}'$ (going down). Set their sum to zero and substitute your expression for $\mathbf{I}'$.

🔑 Show Answer

Node 1: $\mathbf{V}_1 \left[\frac{1}{\mathbf{Z}_1} + \frac{1}{\mathbf{Z}_2}\right] - \mathbf{V}_2 \left[\frac{1}{\mathbf{Z}_2}\right] = \mathbf{I}$
Node 2: $\mathbf{V}_1 \left[\frac{1 - k}{\mathbf{Z}_2}\right] - \mathbf{V}_2 \left[\frac{1 - k}{\mathbf{Z}_2} + \frac{1}{\mathbf{Z}_3}\right] = 0$
